# Employee Management System

This is a web-based application developed using Python Flask and SQLite.

## Modules & Features

### Admin Module
- Admin Login & Dashboard
- Manage Employee Details (Add, Update, Delete)
- Attendance Management (View & Manual Entry)
- QR-Based Attendance Monitoring
- Punch In / Punch Out Monitoring
- Work Report Monitoring
- Leave Approval / Rejection
- Salary Management
- Notice Board Management

### Employee Module
- Employee Login & Profile Management
- Manual Attendance Marking
- QR-Based Attendance
- Punch In / Punch Out
- Work Report Submission
- Apply Leave / My Leave Status
- View Salary Details
- View Notices

### Other Features
- Role-Based Access Control
- Wi-Fi Based Attendance Restriction
- Reports Export (CSV, PDF)

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
